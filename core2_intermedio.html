Crunch Abdominals – Intermedi

COM FER-HO PAS A PAS:

  1. Col·loca un obstacle una mica més gran davant teu ( foam roller o el que vulguis).
  2. Estira't boca amunt i aixeca les cames, mantenint els genolls lleugerament flexionats.
  3. Col·loca les mans darrere
  4. Activa el core i eleva el tronc una mica més que al nivell principiant.
  5. Porta les cames cap a un costat de l'obstacle i després cap a l'altre.
  6. Mantenir el moviment controlat i el core actiu en tot moment.

SÈRIES:

  • 3 sèries de 10–16 repeticions (comptant dreta + esquerra com 1).
  • Descans de 30–45 segons entre sèries.

CONSELLS:

  • Evita arquear l'esquena.
  • Controla el moviment de les cames, no les deixis caure ràpidament.
